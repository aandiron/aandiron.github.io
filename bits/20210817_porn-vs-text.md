# Pornographic Pictures With G-Rated Descriptions
You may know that best practices for web design these days call for a text description of any image to be included in the source code for the web page. Among other techy things, this improves accessibility for people with disabilities. It occurred to me recently that there is a whole set of possible images that someone looking at a visual of the picture would immediately recognize as unequivocal pornography, yet the same image could be legitimately described with a sentence of text that might be read without offense by even the most retrograde pearl-clutching family-values caveman. G-rated descriptions of porn, that is.

Like:
* "Picture of woman enjoying banana."
* "Picture of woman in shorts on a bicycle."
* "Picture of man in jeans standing over a motorcycle."
* "Picture of woman in halter top eating a melting ice cream cone."

It's a fun game, in that nerdy, puzzle-solving, Jeopardy-watching, more-like-"fun", kind of way. Though with a pleasant slight twist of indecency added on top. Maybe you can come up with a favorite of your own? 

But it also has some low-impact implications for all those image descriptions out there on the web. What if someone making use of accessibility features on a web site is _looking_ for pornography? Should the image descriptions for porn be _trying_ to titillate more? Should the text be trying to capture what makes an image _hot_? Or is it more fun to just let the imagination run wild from a description that's pretty tame? This stuff isn't going to make or break accessibility on the internet, but maybe somewhere _someone_ should be asking this question.
