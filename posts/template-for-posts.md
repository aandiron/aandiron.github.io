# A two-hash title with a date

Order for creating a post:
1. In github repo, edit posts.md file. Add link for new post with appropriate file name. Select it to open new postfile.
2. Vexplore to open this template file. Copy format into new post file.
3. Vexplore to open list of things to post. Copy content into new post file. Edit other formate elements of post file. Save post file.
4. Git _add_ and commit new post file. Push to github. Check github pages site to make sure you didn't forget anything.
5. Open new post file with local editor. Copy and paste into https://m2m.pqsec.org/. 
	* Make sure to put in both the title AND include the # title in the markdown body content.
	* Add integration token from stored passwords.
6. Log into medium (through email, so log into email too). Check and publish story.
7. Add to appropriate lists on Medium.
